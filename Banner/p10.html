css-3 radial gradient

in radial gradient color emerge from a single point and smoothly spread outward in a circular or elliptical shape rather than fading from on color to another in a single direction as with linera gradient .the basic syntax of creating radial gradient can be given with

syntax: radial-gradient(shape size at position ,color-stop1,color stop-2,...

the gradients of the radial-gradient() function has the following meaning

  1. position-specifies the startting point of the gradient, which can be specified in units(px, em, or percentage) or keyboard(left, bottom,etc)...
  2. shape-specifies the shape of gradient of the gradient's ending shpae. it can be circle or ellipse
  3. size-specifies the size of the gradient ending shape . the default is farthest side
