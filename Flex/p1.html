CSS FLEX

the flexbox or flexible box model in CSS is a one dimentional layout model that has flexiable and efficient layouts with distributed spaces.

flexiable box layout mdule, makes it easier to design flexiable responible layout structure without usign float or positioning

properties for the parent (flex container)

display: flex;

flex-direction: row;

flex-wrap: wrap;

flex-flow: row-wrap; (flex-flex= fle-direction +flex- wrap)

justify-content:center;

align-items:center;

align-content:space-between;

column-gap:20px;

row-gap:20px;

gap:20px;

properties for the children(flex item)

flex:1; or flex-grow: 1;

flex-shrink:3;

order:2;

align-self:center;
